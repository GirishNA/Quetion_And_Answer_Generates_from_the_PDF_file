To run:

1) Run Supersense & POS tagger.
2) Run QuestionAsker.java for generating knowledge level questions.
3) For other types of questions, such as comprehension level question etc, just open the folder which is named after the type of question and run the specific file.

All jar files have been included in the lib folder. Import them. 
Refer to the readme folder inside questiongeneration for more details.
